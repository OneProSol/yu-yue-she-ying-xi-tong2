### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Flask`、`Python`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`Python 3.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、摄影师、管理员。

用户功能：1.登录、2.注册、3.摄影师作品、4.摄影套餐、5.摄影师、6.个人中心

摄影师功能：1.登录、2.摄影套餐管理、3.我的摄影集、4.摄影编辑、5.预约管理

管理员功能：1.登录、2.用户管理、3.摄影师管理、4.类别管理
